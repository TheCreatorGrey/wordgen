Word Generator for Conlangs

Here's how this works: The generator will go through every comma separated value in the "structure" input below. If the value matches the letter ID of any of the groups in the section below, it will pick a random value from that group. Otherwise, it will interpret it literally and add it to the word. This process will be repeated for the number of times specified in the input below (amount). To add a group, enter a letter ID and alias (name) into the corresponding input boxes, then press "+ Add". Have fun :) This is a modified re-creation of the AwkWords tool.
